## 📌 Sobre o projeto

Aplicação da **técnica Pomodoro** (ciclos de foco de 25 min intercalados com pausas) num timer bonito e funcional. O anel SVG mostra o progresso, o fundo muda de cor conforme o modo e as estatísticas ficam salvas no navegador.

## ✨ Funcionalidades

- ⏳ Anel de progresso circular animado (SVG)
- 🎯 Três modos: **Foco (25')**, **Pausa curta (5')** e **Pausa longa (15')**
- 🎨 Cor do tema muda conforme o modo
- 🔔 Aviso sonoro ao terminar (Web Audio API)
- 📊 Contador de pomodoros e minutos focados (salvo via `localStorage`)
- ⌨️ Atalho: **barra de espaço** inicia/pausa
- 🪧 Título da aba mostra o tempo restante

## 🛠️ Tecnologias

HTML5 · CSS3 (SVG, variáveis) · JavaScript (setInterval, Web Audio, localStorage)
